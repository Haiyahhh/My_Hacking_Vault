# Personal CyberSec Vault
This is where I store my notes when doing the CTF challengers as well as whenever I research new topic on the Internet.
This serves as the knowledge database whenever I need to understand some concepts that I may forgot.

# How to use it?
Go down the [[000 - Global Index]], choose the correct categories and you may find what you're looking for.

# Why is the vault formulated this way?
This vault format is something I found on the Internet when I first try to create a new vault. The original credits belong to [TrustSec](https://www.trustedsec.com/blog/obsidian-taming-a-collective-consciousness) team. 
Essentially, it solves the problem of a notes may belong to multiple category. For example, a note about a HackTheBox machine may belong to both the category of `Active Directory` and `Web Exploitation`.
